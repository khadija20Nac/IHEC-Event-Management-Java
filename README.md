IHEC Event & Room Booking Management System

Java | OOP | MVC | UML

Application Java développée dans le cadre du module de Programmation Orientée Objet (POO).
Ce projet permet de gérer les événements universitaires et la réservation des salles à l’IHEC Carthage, avec un système de validation et de prévention des conflits.

##Fonctionnalités principales

Authentification des utilisateurs

Gestion des événements (création, modification, suppression)

Réservation des salles avec vérification automatique des conflits

Système de validation par l’administrateur

Gestion des rôles : Étudiant, Responsable de club, Administrateur

Affichage du planning global

## Concepts POO appliqués

Encapsulation

Héritage

Polymorphisme

Architecture MVC

Modélisation UML complète

 Technologies utilisées

Java

Java Swing (GUI)

UML (StarUML / draw.io)

IntelliJ IDEA / Eclipse
